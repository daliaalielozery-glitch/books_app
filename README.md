# Bookia App 📚 ✨

A modern Bookstore Application built with Flutter, featuring a secure and elegant authentication flow.

## Tech Stack 🛠️

This project utilizes a professional development workflow with the following tools:

- **UI Scaling:** `flutter_screenutil` (Pixel-perfect responsiveness).
- **Asset Generation:** `flutter_gen` (Type-safe access to images and fonts).
- **Networking:** `dio` (Robust API requests).
- **Localization:** `easy_localization` (Multi-language support).
- **SVG Support:** `flutter_svg` (High-quality vector graphics).
- **Custom Fonts:** `DM Serif Display` for a premium look.
- **Native Splash:** `flutter_native_splash` for a smooth app start.

## Getting Started 🚀

1. Clone the repository.
2. Run `flutter pub get`.
3. Generate assets: `flutter pub run build_runner build`.
4. Run the app: `flutter run`.
