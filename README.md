# Bookia App 📚 ✨

A modern Bookstore Application built with Flutter, featuring a complete and secure authentication flow with a pixel-perfect UI.

## Tech Stack 🛠️

- **Framework:** Flutter
- **UI Scaling:** `flutter_screenutil` (Responsive Design)
- **Networking:** `dio` (API Integration)
- **Localization:** `easy_localization` (Arabic & English)
- **SVG Support:** `flutter_svg`
- **Asset Generation:** `flutter_gen`
- **Splash Screen:** `flutter_native_splash`

## Getting Started 🚀

1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/books_app.git`
2. Get dependencies: `flutter pub get`
3. Generate assets: `flutter pub run build_runner build`
4. Run the app: `flutter run`
